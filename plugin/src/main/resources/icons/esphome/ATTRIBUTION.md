# Icon attribution

The `*.svg` files in this directory are a curated subset of
[Material Design Icons](https://pictogrammers.com/library/mdi/) by Pictogrammers,
licensed under the **Apache License 2.0**.

They are resized to 16×16 (viewBox preserved) and recolored to the IDE's label
foreground at runtime (`EntityIcons`). Only a small, type/device-class-mapped
subset is bundled; arbitrary `icon: mdi:*` values are not rendered.
